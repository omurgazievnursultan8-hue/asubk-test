# Decision Log

A lightweight record of significant decisions, so future-you remembers *why*
something is the way it is. One file per decision: `NNNN-short-title.md`
(e.g. `0001-use-postgres.md`). Copy the template below.

## Index
| # | Date | Decision | Status |
|---|---|---|---|
| _none yet_ | | | |

---

## Template

```markdown
# NNNN — <title>

- **Date:** YYYY-MM-DD
- **Status:** proposed / accepted / superseded
- **Decider(s):** who

## Context
What's the situation / problem forcing a decision?

## Decision
What did we decide to do?

## Consequences
Trade-offs, what this enables, what it costs.

## Alternatives considered
What else was on the table, and why it lost.
```
