### Таблица
| Колонка | Содержимое |
|---------|-----------|
| Просмотр | иконка-стрелка → `/manage/debtor/{id}/view` |

## Итоги P1 (что важно НЕ потерять в новой системе)

1. **Фильтры списка** — поиск по ФИО/названию + Отрасль + Район.
2. **Гео-иерархия заёмщика** — Область → Район → Аильный округ → Село.
3. **Связь с физлицами** — Гл. бухгалтер и Руководитель ссылаются на `/person/{id}`.
4. **Классификаторы заёмщика** — Группа, Вид, Отрасль, Статус акта сверки; меняются
   отдельными кнопками-диалогами прямо из карточки.
5. **Действия по кредитам из карточки** — Расчёт / Акт сверки / Претензия / Сведения /
   Добавить кредит·транш·реструктуризацию — всё завязано на дату.
6. **Три типа «кредита»** в одном потоке: кредит (1), транш (2), реструктуризация (3).
7. **Печатная форма** «Извещение (утрата)» прямо из вкладки залога.

**Дефекты-кандидаты (→ `notes/qa-findings.md`):** непереведённые ключи
`label.debtor.act_status`, `Quantity/Estimated value/Collateral value`,
`label.registerAll*`, `label.pleaseFillDate`; мусорная запись «test» в справочнике
видов заёмщика.
