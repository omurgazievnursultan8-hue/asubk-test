**Architecture:** Single-file vanilla JS мокап. Подход «фундамент вперёд»: сначала сквозные контракты (`requiredCover→{req,source}`, матрица ролей), потом фичи группами. **Spec:** `docs/superpowers/specs/2026-07-21-zalog-fix-design.md`. Источник требований: `~/Downloads/prompt-zalog-fix.md`.

## Global Constraints

- **Single-file, vanilla JS.** Без фреймворков. Дизайн-токены `:root` не менять, новых цветов не вводить.
- **Ни одной заглушки.** Запрещены `TODO`, `// не реализовано`, кнопки без обработчика, модалки без сохранения, объявленные-неиспользуемые константы. `COVER_GUARANTEE`/`COVER_GUARANTEE_FX` — реализовать (Р-11), не удалять.
- **Производные не хранятся.** Порог, залоговая стоимость, покрытие, интервал, статус запрета, ранг — считаются в рантайме. Исключение — событийные записи (акты, решения, извещения).
- **Ссылки на нормы — в нотации** `П1 §2.3` / `П2 §2.5` / `П3 §3.5` / `ПОР п.12` / `ПБК п.2.1`. «§2.3» без префикса запрещён.
- **Регламентные числа — только в справочнике** «Регламентные параметры залога», с пометкой `норматив`/`внутренний`.
- **Язык — русский** везде (UI, комментарии, история, тексты триггеров).
- **Не ломать Р-1…Р-9.** Все существующие демо-ветки работают. Финальный прогон включает регрессию.
- **Номера строк плывут.** Все номера в плане — со снапшота 2026-07-21. Перед каждой правкой перепроверять якорь грепом, а не слепо по номеру.
- **Тест-шов** `window.__zt` держать минимальным — только то, что тесты реально дёргают. Он используется тестами → не заглушка.

```js
import { readFileSync } from 'node:fs';
import { dirname, join } from 'node:path';
import { fileURLToPath } from 'node:url';
import { JSDOM } from 'jsdom';

const HERE = dirname(fileURLToPath(import.meta.url));
const HTML = readFileSync(join(HERE, '..', 'zalog.html'), 'utf8');

// Свежий DOM на каждый тест → изоляция (тесты мутируют CREDITS/ITEMS/CONTRACTS).
export function load() {
  const dom = new JSDOM(HTML, {
    runScripts: 'dangerously',
    pretendToBeVisual: true,
    beforeParse(window) {
      window.matchMedia = window.matchMedia || (() => ({ matches:false, media:'', onchange:null,
        addListener(){}, removeListener(){}, addEventListener(){}, removeEventListener(){}, dispatchEvent(){return false;} }));
      window.scrollTo = window.scrollTo || (()=>{});
    }
  });
  const win = dom.window;
  if (!win.__zt) throw new Error('window.__zt seam missing — скрипт не выполнился или шов отсутствует');
  return { dom, win, zt: win.__zt };
}

let passed = 0, failed = 0; const fails = [];
export function test(name, fn){ try { fn(); passed++; } catch(e){ failed++; fails.push(`  ✗ ${name}\n    ${e.message}`); } }
export function eq(a,b,msg){ if(a!==b) throw new Error(`${msg||'eq'}: expected ${JSON.stringify(b)}, got ${JSON.stringify(a)}`); }
export function near(a,b,msg,eps=0.005){ if(Math.abs(a-b)>eps) throw new Error(`${msg||'near'}: expected ~${b}, got ${a}`); }
export function ok(v,msg){ if(!v) throw new Error(msg||'expected truthy'); }
export function no(v,msg){ if(v) throw new Error(msg||'expected falsy'); }
export function has(hay,needle,msg){ if(!String(hay).includes(needle)) throw new Error(`${msg||'has'}: "${needle}" not found`); }
export function hasNot(hay,needle,msg){ if(String(hay).includes(needle)) throw new Error(`${msg||'hasNot'}: "${needle}" unexpectedly present`); }
export function report(){ console.log(`\n${passed} passed, ${failed} failed`); fails.forEach(f=>console.log(f)); if(failed) process.exit(1); }
```
